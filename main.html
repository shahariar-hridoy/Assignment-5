Question-1: What are the different ways to select an element in the DOM?

There are may ways to select an element in the DOM. Like,
1. getElementById("id")
2. getElementByClassName("class-name")
3. getElementByTagName("h1")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML set or get HTML inside the element.
innerText set or get only text inside the element.
textContent set or get all text except HTML.

Question-3: What is event delegation in the DOM?

Event delegation is an event that adding event listener on parent insted of child, and that works for every child.

Question-4: What is event bubbling in the DOM?

Event bubbling means when click on an element, the event first happen on the clicked element then happen on parent element , grand parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

For creating we use: document.createElement('p');
For adding we use: document.name.appendChild('name1');
For remove element : name.remove();
